Создание и сохранение изображения


Цель урока


Освоить приемы создания, открытия, сохранения изображения.

Задачи


- научиться открывать, создавать и сохранять изображения в GIMP;

- научиться задавать размеры изображения;

- познакомиться с готовыми шаблонами графических объектов.


Требования к уровню подготовки обучающихся


Должны знать: способы открытия, создания, сохранения графического файла. Шаблоны изображения. 

Должны уметь: создавать, открывать, сохранять изображения, пользоваться расширенными настройками и шаблонами.

Сохранение изображения


Совет

Следует взять себе за правило, никогда не работать с оригиналом изображения. Поэтому, посмотрев изображение, для дальнейшей работы его лучше сохранить в другой каталог. Для этого: выберите в меню изображения Файл - Сохранить как и в диалоговом окне сохранения укажите нужный каталог, имя файла и его расширение (по умолчанию).

 

Задание 1


1. Запускаем редактор GIMP.

2.  Создаем новое изображение - меню Файл - Создать. В появившемся окне Создать новое изображение задаем характеристики создаваемого изображения. 

Размер изображения - задаем ширину и высоту картинки - 800х600. По стандартным настройкам стороны измеряются в точках (пикселях).

Выбираем Альбом в опции Расположение листа с изображением.

3. Опция Расширенные параметры содержит дополнительные параметры создаваемого изображения.

Разрешение изображения отвечает за то, сколько точек будет располагаться на одном квадратном дюйме изображения.

Устанавливаем Разрешение по X - 300,  Разрешение по Y - 300 (оптимальное количество, достаточное для четкого отображения картинки на экране и для любительской печати изображений. Для профессиональной печати используются более высокие значения Разрешения изображений)

4. Выбираем одну из двух цветовых моделей — Цвета RGB и Серый опции Цветовое пространство. Первый вариант - для создания и редактирования  полноцветных изображений, второй - только черно-белые.

5. Выбираем  один из четырех вариантов фона изображения.   Заполнение - белый.

6. Нажимаем ОК.

7. Сохраняем файл: Файл  - Сохранить, указав в диалоговом окне имя файла, директорию файла, тип файла. Файл сохраняется в оригинальном формате GIMP - xcf.

8. Чтобы сохранить файл под другим именем выбираем меню Файл - Сохранить как.


Задание 2


1. Запускаем редактор GIMP.

2. В выпадающем меню Файл - подменю Создать выбираем Снимок экрана, которая позволяет создать изображение и автоматически заполнить его снимком, сделанным с экрана. При выборе данной опции появляется окно, в котором задаем область захвата — одно окно или весь экран, а также задержку в секундах осуществления захвата изображения.

3. Сохраняем файл в формате jpg: меню Файл - Экспортировать как..., в диалоговом окне задаем имя и директорию файла, тип файла выбираем из предложенного списка (GIMP версии 2.8). В версии 2.6. выбираем Файл - Сохранить как, задаем имя, директорию, выбираем формат файла из предложенного списка.


Задание 3


1. Создаем новое изображение, выбрав варианты Кнопки, Логотипы, Текстуры, Темы веб-страниц из предлагаемого списка. В появившемся окне настройки данного шаблона   задаем текст, размер шрифта, тип шрифта, цвет фона. По окончании настройки всех параметров нажимаем кнопку OK.

2. Сохраняем файл в формате png: Файл - Экспортировать как... (для форматов png и  jpg в версии GIMP 2.8)
