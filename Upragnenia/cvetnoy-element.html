Цветной элемент на черно-белом фото


Цель урока


Создать черно-белую фотографию с ярко выделяющимся цветным элементом.

Задачи


- Научиться работать с инструментом Умные ножницы.


Требования к уровню подготовки обучающихся


Должны знать: инструменты Умные ножницы, Обесцвечивание.

Должны уметь: выполнять выделение с использованием подходящих инструментов выделения, работать с инструментом Цвет.


Учебный материал


Ознакомьтесь, пожалуйста, с разделами справки.

Инструмент Умные ножницы


Этапы работы



На уроке мы научимся создавать интересный эффект выделения объекта с помощью цвета на черно-белом изображении.

1. Запускаем редактор  GIMP.

2. Открываем файл с исходным изображением: Меню - Файл - Открыть. Учебник GIMP\Upragnenia\Imgup\frukti.jpg

3. Выбираем инструмент Умные ножницы и  выделяем необходимый участок фотографии.

Данный инструмент выделяет часть изображения с помощью создания множества контрольных точек, с функцией определения границы выделяемого участка на фотографии. И чем больше  создается таких точек, тем более качественным и лучшим будет выделение.

В процессе работы с инструментом Умные ножницы  расставляются контрольные точки по границе выделяемого объекта. Также в процессе работы с инструментов  можно добавлять еще дополнительные точки, просто щелкая по границе выделяемого объекта и включая в выделение нужную часть изображения.

2. В параметрах инструмента Умные ножницы поставим галочку напротив пункта Растушевать края с параметром 10. Нажмите кнопку Enter на клавиатуре.

3. Инвертируем выделение Меню - Выделение - Инвертировать (Ctrl+I). 

4. Обесцветим инвертируемое выделение с помощью инструмента Меню - Цвет Обесцвечивание.


5. Снимаем выделение:  Выделение - Снять.

6. Результат: за несколько простых шагов мы получили  очень интересный эффект на фотографии: цветной элемент на черно-белом фоне.
