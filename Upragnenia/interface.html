Интерфейс редактора GIMP


Цель урока


Знакомство с интерфейсом редактора.

Задачи


- познакомиться с элементами окна: выпадающее меню, диалоговые окна, рабочая область. панель инструментов.

Требования к уровню подготовки обучающихся


Должны знать:  элементы окна редактора.

Должны уметь: настраивать интерфейс приложения.


Учебный материал


Ознакомьтесь, пожалуйста, с разделами справки.

Интерфейс редактора


Этапы работы


1. Запустите приложение.

2. Ознакомьтесь с интерфейсом программы. Изучите элементы рабочего окна редактора.

3. Пройдите по ссылке.

4. Создайте копию документа.

5. Заполните текстовые поля.

6. Сохраните файл.

7. Покажите файл преподавателю.
