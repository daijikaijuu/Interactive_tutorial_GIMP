Эффект дождя на фотографии


Цель урока


Научиться создавать эффект дождя на фотографии

Задачи


- Научиться работать с фильтрами.


Требования к уровню подготовки обучающихся


Должны знать: возможности применения фильтров, их особенности.

Должны уметь: работать со слоями, фильтрами, создавать разнообразные эффекты.


Учебный материал


Ознакомьтесь, пожалуйста, с разделами справки.

Инструмент Фильтры


Этапы работы



1. Запускаем редактор  GIMP.

2. Открываем файл с исходным изображением: Меню - Файл - Открыть.

3. Создаем новый слой - Дождь.

4. Применяем Фильтры  - Шум - Случайный выбор с параметрами: случайное зерно - 50, случайность - 80, повторов - 30.

 5. Обесцветим изображение: Цвет - Обесцвечивание.

6. Далее "шум" необходимо размыть: Фильтр - Размывание - Размывание в движении. Параметры размывания: длина – 50, угол – 75.

4. Чтобы обострить эффект дождя, используем  Фильтр - Искажение - Рябь. Параметры: сглаживание края  - вкл., ориентация - по вертикали, края - размыть, тип волны - синус, период - 60, амплитуда - 0, сдвиг по фазе  - 5.

5. Уменьшаем прозрачность верхнего слоя до 50%.

6. С помощью инструмента Кисть подчищаем изображение.
