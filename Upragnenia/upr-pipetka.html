Работа с инструментом Пипетка


Цель урока


Научиться получать цвет из изображения.

Задачи


- освоить приемы работы с инструментом Пипетка;

- научиться настраивать режимы работы с инструментом.



Требования к уровню подготовки обучающихся


Должны знать:  кодирование цвета, свойства и назначение инструмента Пипетка.

Должны уметь: работать с инструментом Пипетка, настраивать режимы работы, менять свойства инструмента.


Учебный материал


Ознакомьтесь, пожалуйста, с разделами справки.

Инструмент Пипетка



Ход урока



1. Откройте редактор.

2. Загрузите произвольное изображение.

3. На панели инструментов выберите инструмент Пипетка, ознакомьтесь со свойствами инструмента.

4. Получите десятичный, шестнадцатеричный коды и процентное соотношение каждого цвета  цвета для 5 различных участков изображения.

5. Заполните таблицу "Информация о цвете"
