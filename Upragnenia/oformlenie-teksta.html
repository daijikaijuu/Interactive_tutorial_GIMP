Создание текста по контуру

 


Цель урока


Научиться создавать текст по заданному контуру.


Задачи


- изучить порядок и  приемы работы с текстом;

- научиться создавать выразительный текст.


Требования к уровню подготовки обучающихся


Должны знать: инструменты и возможности редактора.

Должны уметь: работать с инструментом Текст, создавать текст по контуру.


Этапы работы


Учебный материал. Ознакомьтесь, пожалуйста, с разделами справки.


Инструмент  Текст

Инструмент  Контуры


Основные приемы работы


Рассмотрим некоторые приемы работы с текстом: порядок создания текста вдоль заранее заданного пути.

1. Запускаем GIMP. Создаем новое пустое изображение. Размеры  примерно 640*480 пикселей. Таким образом, мы получаем полностью пустое изображение, которое состоит из одного слоя – Фон.

2. Сначала необходимо создать линию, вдоль которой впоследствии будет выровнен текст. Для этого нам понадобится инструмент Контуры на панели инструментов. Выбираем данный инструмент, затем рисуем линию любой формы, например,  волнообразную.

Подсказка

Для создания округлых форм, а не ломанных, необходимо воспользоваться касательными линиями, расположенными у каждой узловой точки формы контура.

3. Создадим сам текст. Выбираем инструмент Текст на панели инструментов и набираем любую фразу. Размер букв задаем небольшим, чтобы на сгибах волны буквы не накладывались друг на друга.

4. Для выравнивания формы текста вдоль пути контура, сначала переходим на панель Контуры  и выделяем единственную имеющуюся там строчку – Безымянный. Это – наш контур. Для удобства, можно даже включить его видимость при помощи кнопки в левом углу строчки. Затем выбираем на панели инструментов инструмент Текст и щелкаем по уже созданной надписи. В нижней части панели инструментов появляются параметры набранного текста. Нажимаем здесь кнопку Текст по контуру.

В результате – текст останется на месте, а вдоль линии контура появится новый контур – по форме надписи.

5. Получен новый контур, форма которого равна форме надписи, выровненной вдоль первого контура. Теперь нам необходимо превратить этот контур в полноценные буквы. Переходим к панели Слои и выделяем на ней слой Фон. Затем возвращаемся к панели Контуры и выделяем здесь второй контур, который был создан автоматически, название которого совпадает с вашей надписью. Далее необходимо нажать кнопку Контур – Выделение, расположенную на нижнем блоке опций панели Контуры.

6. Мы превратили контуры букв в выделение. Теперь можно как угодно залить выделенный регион для получения изображения букв. Например, можно взять инструмент Кисть и при помощи него закрасить буквы, придав им цвет. Можно взять инструмент Градиент и сделать буквы переливающимися или плавно растворяющимися. Можно заполнить буквы каким-нибудь насыщением при помощи уже известных нам фильтров. На рисунке ниже приведены варианты заполнения выделения по форме букв.

7. После создания текста вдоль пути, оригинал текста можно удалить. Для этого можно удалить слой, содержащий данный текст. Выделение снять.

Итак, мы получили надпись, сформированную вдоль заранее созданного пути.


Вывод

Можно придавать любую форму любому тексту.
