Совет

Следует взять себе за правило, никогда не работать с оригиналом изображения. Поэтому, посмотрев изображение, для дальнейшей работы его лучше сохранить в другой каталог. Для этого: выберите в меню изображения Файл - Сохранить как и в диалоговом окне сохранения укажите нужный каталог, имя файла и его расширение (по умолчанию).

 

Задание 1


Создайте Новый файл с размерами 640х480 px. Расположение листа - Альбом. Цветовое пространство - RGB. Экспортируйте файл Файл - Экспортировать как.. в формат png, указав имя файла Praktika1 и директорию файла (свою папку).


Задание 2


Создайте текстуру Вихрь. Параметры: размер четверти - 200, угол вихря - 90, число повторов для вихря - 8. Сохраните файл в свою папку в оригинальном формате GIMP. Имя файла - Praktika2.


Задание 3


Создайте Логотип - Текстуризованный.  Параметры: текст "Графический редактор GIMP", кегль шрифта - 200, тип мозаики - восьмиугольники, цвет фона - FFFFAA, начальный цвет - 20F350, конечный цвет - ECECEC. Сохраните файл в формате jpg. Имя файла  Praktika3.


Задание 4


Создайте Новое изображение. Размер 800х600. Расположение листа - Портрет. Разрешение по оси Х - 100, разрешение по оси У - 80. Цветовое пространство - Серый.  Цвет фона - Белый. Комментарий - Новое изображение. Настройте параметры страницы: ориентация - альбомная, поля  - 2. Экспортируйте файл в свою папку в формате gif, указав имя файла -  Praktika4.


Задание 5


Создайте Новое изображение - Шаблон - Web banner huge 728x90. Разрешение 72х72, Цветовое пространство - RGB,Фон - прозрачный. Экспортируйте в свою папку в формате tiff. Имя файла - Praktika5.
