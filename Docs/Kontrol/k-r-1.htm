Вопрос 4. Если мы сохраняем изображение в качестве проекта до следующего редактирования в GIMP, необходимо выбирать формат:

Ответ 1. .jpg
Ответ 2. .xcf
Ответ 3. .bmp
Ответ 4. .gif

Вопрос 5. Укажите группу, в которой содержатся расширения только графических файлов.

Ответ 1. bmp, jpg, gif, tif, png
Ответ 2. bmp, jpg, gif, swf, png
Ответ 3. bmp, htm, jpg, gif, swf
Ответ 4. bmp, jpg, gif, com, flx

Вопрос 6. Если нам нужно сделать небольшое по размеру изображение, например, открытку или чью-либо фотографию, а затем распечатать его, то целесообразно использовать формат:

Ответ 1. .tif
Ответ 2. .xcf
Ответ 3. .jpg
Ответ 4. .bmp

Вопрос 7. Выберите верное утверждение. Формат .bmp

Ответ 1. Поддерживается всеми графическими редакторами и применяется для хранения векторных изображений.
Ответ 2. Поддерживается всеми графическими редакторами и применяется для хранения растровых изображений
Ответ 3. Оригинальный формат Adobe Photoshop, способен хранить до 16 млн. цветов.
Ответ 4. Универсальный несжимаемый формат Windows.

Вопрос 8. Выберите верное утверждение. Формат .gif - это...

Ответ 1. Формат растровых графических файлов, позволяет хорошо сжимать файлы, в которых много однородных заливок.
Ответ 2. Формат векторных графических файлов, позволяет хорошо сжимать файлы, в которых много однородных заливок
Ответ 3. Формат векторных графических файлов, цветное изображение может быть записано только в режиме 256 цветов.
Ответ 4. Несжимаемый формат растровых графических файлов.

Вопрос 9. Если мы создаем большое изображение для широкоформатной печати, например, баннер или вывеску, целесообразно использовать формат:

Ответ 1. .bmp
Ответ 2. .xcf
Ответ 3. .jpg
Ответ 4. .gif

Вопрос 10. Разрешение экрана измеряется в

Ответ 1. дюймах
Ответ 2. точках на дюйм
Ответ 3. миллиметрах
Ответ 4. пикселях

Вопрос 11. Выберите правильный вариант ответа. Применение векторной графики по сравнению с растровой:

Ответ 1.  не меняет способы кодирования изображения;
Ответ 2.  увеличивает объем памяти, необходимой для хранения изображения;
Ответ 3. не влияет на объем памяти, необходимой для хранения изображения, и на качество отредактированных рисунков; 
Ответ 4. уменьшает объем памяти, необходимой для хранения изображения, и сохраняет качество рисунка при его масштабировании.

Вопрос 12. Разрешающая способность экрана в графическом режиме определяется:
