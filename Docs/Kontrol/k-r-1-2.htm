Вопрос 1. Основной объект, с которым работает GIMP.

Ответ 1. слой
Ответ 2. канал
Ответ 3. изображение
Ответ 4. окно

Вопрос 2. Выберите верное утверждение. Формат .bmp -  ...

Ответ 1. Универсальный несжимаемый формат Windows.
Ответ 2. Оригинальный формат Adobe Photoshop, способен хранить до 16 млн. цветов.
Ответ 3. Поддерживается всеми графическими редакторами и применяется для хранения векторных изображений.
Ответ 4. Поддерживается всеми графическими редакторами и применяется для хранения растровых изображений

Вопрос 3. Выберите правильный вариант ответа. Применение растровой графики по сравнению с векторной ...

Ответ 1.  не меняет способы кодирования изображения;
Ответ 2. увеличивает объем памяти, необходимой для хранения изображения;
Ответ 3. не влияет на объем памяти, необходимой для хранения изображения, и на качество отредактированных рисунков; 
Ответ 4. уменьшает объем памяти, необходимой для хранения изображения, и сохраняет качество рисунка при его масштабировании.

Вопрос 4. Цветное с палитрой из 256 цветов растровое графическое изображение имеет размер 10х10 точек. Какой объем памяти в байтах займет это изображение?

Ответ 1. 100
Ответ 2. 800
Ответ 3. 10
Ответ 4. 80

Вопрос 6. Какая цветовая модель представлена на рисунке?

Ответ 4. LAB

Вопрос 7. Выберите неверное утверждение.

Ответ 1. Растровые изображения масштабируются с потерей качества.
Ответ 2. Один из недостатков векторной графики - большой размер графических файлов.
Ответ 3. Векторные изображения масштабируются без потери качества.
Ответ 4. Цветовая модель RGB - является аддитивной.

Вопрос 8. Укажите вид компьютерной графики, получающийся в результате сканирования изображения.

Ответ 2. трехмерная
Ответ 3. растровая
Ответ 4. фрактальная

Вопрос 9. Выберите верное утверждение. Формат JPG - это ...

Ответ 1. Универсальный формат, рекомендуется хранения растровых картинок фотографического качества.
Ответ 2. Оригинальный формат для хранения растровых картинок фотографического качества.
Ответ 3. Универсальный формат, рекомендуется хранения векторных изображений.
Ответ 4. Универсальный формат, использует сжатие без потерь.

Вопрос 12. Ученик создал Web-страницу в Интернете и задал сначала фон красным, а потом изменил его на синий.
Как изменился вид атрибута bgcolor для тэга <body>?

Ответ 1. с <body bgcolor="#00FF00" на <body bgcolor="#FFFFFF">
Ответ 2. с <body bgcolor="#FF0000" на <body bgcolor="#0000FF">
Ответ 3. с <body bgcolor="#0000FF" на <body bgcolor="#00FF00">
Ответ 4. с <body bgcolor="#000000" на <body bgcolor="#0000FF">

Вопрос 15. Выберите верное утверждение. Формат .gif - это ...

Ответ 1. Формат растровых графических файлов, позволяет хорошо сжимать файлы, в которых много однородных заливок.
Ответ 2. Формат векторных графических файлов, позволяет хорошо сжимать файлы, в которых много однородных заливок.
Ответ 3. Формат векторных графических файлов, цветное изображение может быть записано только в режиме 256 цветов.
Ответ 4. Несжимаемый формат растровых графических файлов

Вопрос 16. Выберите верное утверждение. Формат .png - это ...

Ответ 1. Несжимаемый оригинальный формат для Интернета.
Ответ 2. Универсальный формат, использует сжатие без потерь, реализует основную идею WWW - одинаковое отображение информации независимо от аппаратуры пользователя, глубина цвета ограничена до 8 бит.
Ответ 3. Универсальный формат, использует сжатие без потерь, реализует основную идею WWW - одинаковое отображение информации независимо от аппаратуры пользователя.
Ответ 4. Оригинальный формат с хорошим сжатием и любой глубиной цвета, вплоть до 48 бит.
